This is the first step to your roblox journey!


Navigation

    Hold RMB and drag your mouse to look around

    Use WASD and the mouse wheel to move

    LMB on an object to select it

    RMB on an object to select and open the context menu

Tools

    The select tools lets you select with LMB

    The move tool lets you move objects

    The rotate tool lets you rotate objects around the pivot point

    The scale tool lets you scale objects
